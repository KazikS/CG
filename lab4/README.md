# Лабораторная работа №2: Камера и освещение (Вариант 3)

Реализация камеры с матрицей трансформации и освещения по модели Блинн-Фонга с рассеянным, направленным и прожекторными источниками света.

## Сцена

- Пол (серый)
- Куб (синий)
- Пирамида (красная)
- Сфера (зелёная)
- Куб (белый, на заднем плане)
- 2 прожекторных источника света

## Возможности

- **Камера:** матрица трансформации (положение + ориентация)
- **Управление:** WASD — перемещение, мышь (ЛКМ) — вращение, Q/E — вверх/вниз
- **Освещение:** рассеянное (ambient), направленное (directional), прожекторные (spotlight)
- **Блинн-Фонг:** диффузный + спекулярный компоненты для каждого источника
- **Прожекторы:** затухание по закону обратных квадратов, гладкие края (inner/outer cutoff)
- **SSBO:** shader-storage буферы для прожекторных источников
- **Материалы:** albedo, specular color, shininess для каждого объекта
- **UI (ImGui):** управление параметрами всех источников света в реальном времени

## Требования

- Windows 10/11
- Visual Studio 2022 (компонент C++ Desktop)
- CMake >= 3.20
- Ninja
- Vulkan SDK

## Сборка и запуск

Открыть **x64 Native Tools Command Prompt for VS 2022**:

```bash
cd lab2
cmake --preset debug
cmake --build build-debug --parallel
glslc shaders/shader.vert -o shaders/shader.vert.spv
glslc shaders/shader.frag -o shaders/shader.frag.spv
.\build-debug\testbed\testbed.exe
```
